Responsive Grids with Bootstrap

What's the point?

It's important that users are able to use applications across all of their devices, providing a seamless experience. A responsive grid system can help us design these seamless experiences easily, quickly, and in a standard, maintainable way.

What do you mean by 'responsive'?

When I talk about ‘responsive’, I’m talking about creating application design that is user friendly across all screen sizes.

Think about all the devices users have today: desktop, laptop, tablets, phablets, phone, wearables. All of in an array of various screen sizes.

Our goal is to create a design that can respond to any screen size and provide a good user experience.

Who uses Bootstrap anyway?

Bootstrap was developed by Twitter. But it's not just Twitter that uses it - there's a whole bunch of places that use Bootstrap. Bootstrap has a site dedicated to showing off sites that use Bootstrap.

You probably visit sites that use it every day without realizing it. Check out this one..

What is a grid system?

Generally, a grid system is a set of measurements a designer can use to align and size objects within the given format.

Bootstrap provides its own grid system of rows and columns that we can use to quickly create responsive layouts for all screen sizes.

So what's this whole thing about?

We're going to be talking about applying Bootstrap classes to HTML containers (in this case divs.) to achieve a desired layout.
