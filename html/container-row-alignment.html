2. Grid Alignment

Content should be placed within columns, and only columns may be immediate children of rows.

This is just some content in a container - but not in a grid!

<--- Notice how it aligns in the container?

Containers use padding to align content.

We're in a row now.

<-- Check out how the alignment is not the same as the container content.

This is because rows use a negative margin in order to use the whole width of the container.

For this reason, only columns should be children of rows.
