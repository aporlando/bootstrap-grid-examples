3.5. Offsetting Columns

Offsets allow us to place columns at any available column point in the grid. Breakpoint prefixes apply to offsets as well. Try shrinking your browser to see how the .col-md-offset-* classes don't apply to smaller screens.

.col-md-4
.col-md-4 .col-md-offset-4
.col-md-3 .col-md-offset-3
.col-md-3 .col-md-offset-3
.col-md-6 .col-md-offset-3

The Code

<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4 col-md-offset-4">.col-md-4 .col-md-offset-4</div>
</div>

    <div class="col-md-3 col-md-offset-3">.col-md-3 .col-md-offset-3</div>
</div>
